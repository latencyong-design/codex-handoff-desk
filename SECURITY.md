# Security Policy

## English

Codex Handoff Desk reads local Codex session JSONL files and generates local Markdown/HTML outputs.

Privacy boundary:

- Generated output may still contain sensitive project context.
- Redaction is best-effort, not a complete secret scanner.
- Review generated handoffs before posting them publicly.

Please report security issues through a private GitHub security advisory if available, or open an issue with a sanitized reproduction that does not include real secrets.

## 中文

Codex Handoff Desk 会读取本地 Codex session JSONL，并生成本地 Markdown / HTML 输出。

隐私边界：

- 生成内容仍然可能包含敏感项目上下文。
- 脱敏是 best-effort，不是完整 secret scanner。
- 公开发布 handoff 前请人工检查。

如发现安全问题，请优先使用 GitHub private security advisory；如果只能开 issue，请提供脱敏后的复现信息，不要贴真实密钥。
